Week 3

Advanced CSS

In this third week we went further in depth on CSS and started doing CSS flexbox, it is a layout model that provides an easy and clean way to arrange items within a container. Then to make web pages more dynamic we started to create animations on a web page. You can change as many CSS properties as you want. And you can do this as many times as you wanted with CSS transitions you have the potential to alter the appearance and behavior of an element whenever a state change occurs, such as when it is hovered over, focused on, active, or targeted. Animations within CSS allow the appearance and behavior of an element to be altered in multiple keyframes.
