HTML and CSS

Now that we learned about fundamentals of javascript in this second week we started using HTML, HTML stands for Hyper Text Markup Language. HTML is the standard markup language for creating Web pages. HTML used to structure a web page. HTML consists of a series of elements and all we did was to check what each element does. After doing HTML we did CSS, it stand for Cascading Style Sheets and is a computer language for laying out and structuring web pages.
